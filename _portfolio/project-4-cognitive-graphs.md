## Overview

A hallmark of intelligent behavior is the ability to generalize — to apply knowledge learned in one context to a new but related one. This requires more than memorizing stimulus-response associations; it requires building an internal model of the *structure* of the world. How does the brain represent such abstract structure?

In this project (joint first-authorship with M. Manakov), we investigated whether the rostral ACC encodes a **cognitive map** of latent task structure — and if so, what form that representation takes.

## Key Findings

- The rostral ACC encodes latent task structure in the form of **cognitive graphs**: neural population representations whose geometry mirrors the abstract relational topology of the task.
- These graph-like representations are **not simply derived from sensory input or motor output** — they reflect the animal's learned model of task contingencies.
- The geometry of ACC cognitive graphs **predicts generalization performance**: animals with more faithful neural representations of task structure generalize better to new task variants.
- Representations are **compositional** — the brain appears to combine simpler relational elements to encode more complex task structures.

## Methods & Tools

- Large-scale Neuropixels recordings in rostral ACC
- Graph-theoretic analysis of neural population geometry
- Representational similarity analysis (RSA)
- Task designs with explicit latent relational structure
- Collaboration with S. Druckmann (computational modeling)

## Significance

This work provides evidence that the rostral ACC functions as a substrate for **cognitive map-like representations of abstract task structure** — analogous to hippocampal place maps, but for relational knowledge rather than physical space. It connects prefrontal cortex function to a broader framework of structured, model-based cognition and offers a neural-level account of how flexible, knowledge-guided behavior is supported in the brain.
